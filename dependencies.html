Dependencies

  • SASS / Sass lets you use features that don't exist in CSS yet like variables, nesting, mixins, inheritance and other nifty goodies that make writing CSS fun again. Currently SASS is used in Producer/Lifeguard and Design Style Guides
  • jQuery / jQuery is a fast, small, and feature-rich JavaScript library. Used in all our products.
  • Bootstrap / Bootstrap is the most popular HTML, CSS, and JS framework. Currently used in Courier.
  • DataTables / DataTables is table plug-in for jQuery. Widely used for our all products.
  • Select2 / Select2 is a jQuery based replacement for select boxes. It supports searching, remote data sets, and infinite scrolling of results.
